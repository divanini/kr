1003. Можно ли из бревна, имеющего диаметр поперечного сечения D, выпилить квадратный брус шириной A?

В поле "Диаметр бревна" пользователь вводит диаметр бревна в метрах, а в поле "Ширина квадратного бруса" - желаемую ширину квадратного бруса в метрах.

При нажатии на кнопку "Рассчитать" вызывается функция calculate(). Внутри функции calculate(), мы сначала извлекаем значения из полей ввода.

После этого мы проверяем, являются ли введенные значения числами. Если пользователь ввел нечисловые значения, мы выводим сообщение об ошибке.

Если значения являются числами, мы используем деление для расчета количества квадратных брусьев, которые можно получить из бревна. Мы используем Math.floor() для округления результата в меньшую сторону, так как нельзя получить дробное количество брусьев.

Затем результат помещаем в элемент с id="result"
